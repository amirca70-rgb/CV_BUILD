منصة ويب احترافية لإنشاء سيرة ذاتية بمعايير دولية مدعومة بالذكاء الاصطناعي.

```

```

---

## 🚀 تشغيل المشروع

### الطريقة 1: VS Code Live Server (موصى بها)
1. افتح المجلد `cv-builder` في VS Code
2. ثبّت إضافة **Live Server** إذا لم تكن مثبتة
3. انقر بزر الماوس الأيمن على `index.html` → **Open with Live Server**

### الطريقة 2: Python HTTP Server
```bash
cd cv-builder
python -m http.server 8080
# افتح: http://localhost:8080
```

### الطريقة 3: Node.js
```bash
npm install -g serve
cd cv-builder
serve .
```

## 🔑 إعداد مفتاح Anthropic API

لاستخدام ميزات الذكاء الاصطناعي:

1. اذهب إلى [console.anthropic.com](https://console.anthropic.com)
2. أنشئ حساباً واحصل على مفتاح API
3. في صفحة المنشئ، عند الضغط على أي زر AI سيظهر نافذة لإدخال المفتاح
4. أدخل المفتاح وسيُحفظ تلقائياً على جهازك (localStorage)

> ⚠️ **مهم:** المفتاح يُحفظ محلياً على جهازك فقط ولا يُرسل لأي خادم خارجي.

## ✨ الميزات

### المعايير المدعومة
| المعيار | الدول | المميزات |
|---------|-------|----------|
| 🇪🇺 Europass | 32 دولة أوروبية | جدول CEFR للغات، صفحة مهارات |
| 🇨🇦 Canadian | كندا | ATS-optimized، إنجازات كمية |
| 🇬🇧 British | المملكة المتحدة | صفحتان، ملف شخصي |
| 🇺🇸 American | الولايات المتحدة | صفحة واحدة مكثفة |

### ميزات الذكاء الاصطناعي
- **توليد الملخص المهني** — أدخل وصفاً بالعربية أو الإنجليزية
- **صياغة الخبرات** — تحويل المهام العادية لنقاط احترافية
- **اقتراح المهارات** — مقترحات ذكية حسب المسمى الوظيفي
- **حساب ATS Score** — تقييم توافق السيرة مع أنظمة الفرز
- **تحسين مخصص** — طلب تحسينات محددة بالنص
